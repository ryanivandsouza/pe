## Getting Started

### Prerequisites

* Docker: Refer [official page for installation](https://docs.docker.com/v17.09/engine/installation/)

### Usage

#### Linux
```sh
git clone https://github.com/ryanivandsouza/pe.git
cd pe/rust/euler
./scripts/run.now.sh
```

#### Windows
```sh
git clone https://github.com/ryanivandsouza/pe.git
cd pe/rust/euler
./scripts/run.now.ps1
```

### Build

#### Linux

```sh
git clone https://github.com/ryanivandsouza/pe.git
cd pe/rust/euler
./scripts/build.now.sh
```

#### Windows
```sh
git clone https://github.com/ryanivandsouza/pe.git
cd pe/rust/euler
./scripts/build.now.ps1
```
